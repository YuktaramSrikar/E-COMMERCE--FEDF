# Product Images

Add your handloom product images to this folder with the following names:

1. `banarasi-saree.jpg` - Banarasi Handloom Saree
2. `pochampally-dupatta.jpg` - Pochampally Ikat Dupatta
3. `cotton-stole.jpg` - Handwoven Cotton Stole
4. `kanchipuram-saree.jpg` - Kanchipuram Silk Saree
5. `ajrakh-print.jpg` - Ajrakh Block Print Fabric
6. `sambhalpuri-saree.jpg` - Sambhalpuri Handloom Saree

Images should be at least 400x250 pixels in size for optimal display.

If images are not present, the app will fall back to external URLs automatically.
